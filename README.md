# Grayscale Histogram Equalizer

## To Build
```
git clone git@github.com:gksbrandon/grayscale-histogram-equalizer.git --recurse-submodules
cd grayscale-histogram-equalizer
mkdir build
cd build
cmake ..
make
```

## To Run Tests
```
cd build/test
./HistogramEqualizationTests
```

## To Run
```
cd build
./HistogramEqualizationExe
```
